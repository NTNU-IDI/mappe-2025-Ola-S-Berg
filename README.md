# Mappeoppgave-Diary
Diary is an application that lets you create and manage diary entries.
The application is developed as a student project at the Norwegian University of Science and Technology (NTNU).

## Features
Since the latest v1.1.0 release, the following core functionalities are implemented:
- Template functionality
  - Standard entry template
    - A standard diary entry with no special fields, great for everyday use
  - Gym entry template
    - An entry for gym workouts, great for tracking progress
    - Add excercises performed, number of sets, repetitions and weight used
  - Fishing entry template
    - An entry for fishing, great for logging your adventures
    - Add weather conditions, fish caught, location and bait used
- Text-based UI functionality:
  - Create new diary entry
    - Choose between templates (standard, gym or fishing)
    - Choose between- or create a new author upon entry creation
    - Enter a fitting title and category for the entry
    - Write the content of the diary entry
  - View entries
  - Search entries
  - Edit existing entries
  - Delete entries
  - Manage Authors
  - Show statistics
