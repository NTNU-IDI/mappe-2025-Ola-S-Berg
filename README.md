# Mappeoppgave-Diary
Diary is an application that lets you create and manage diary entries.
The application is developed as a student project at the Norwegian University of Science and Technology (NTNU).

## Features
In the latest v0.2.0 pre-release, the following core functionalities are implemented:
- Create a diary entry with the following parameteres:
  - Author
  - Title
  - Content
  - Category

## Tech stack
The application is built using the following technologies:
- Java
- Maven
- Github
